January 8-9, 2026

WHAT IS CS NAVIGATOR?
---------------------
CS Navigator is a chatbot application for Morgan State University Computer
Science students. It helps students with course information, academic advising,
and navigating their degree requirements.


WHAT WE ACCOMPLISHED (JANUARY 8-9, 2026)
========================================

1. CHAT HISTORY & MEMORY
------------------------
   What it does: The chatbot now remembers your previous conversations and
   can answer follow-up questions based on context.

   Features:
   - All chat history is saved to the database
   - When you ask a follow-up question, the bot remembers what you talked about
   - Conversations are organized by sessions in the sidebar
   - You can search through old conversations
   - You can archive conversations you don't need anymore

   How it works: When you send a message, the bot looks at your recent
   conversation history to understand context and give better answers.


2. DYNAMIC SUGGESTED QUESTIONS
------------------------------
   What it does: The welcome screen shows the most frequently asked questions
   by all students, updated automatically.

   Features:
   - Questions are pulled from actual chat history
   - Shows what other students are commonly asking
   - Falls back to default questions if not enough data
   - Updates automatically as more students use the app


3. VOICE INPUT (MIC BUTTON)
---------------------------
   What it does: You can now speak your question instead of typing.

   How to use:
   1. Click the microphone button next to the input field
   2. Speak your question clearly
   3. The text will appear in the input box
   4. Press Send or Enter to submit

   Note: Works best in Chrome or Edge browsers.


4. DEGREEWORKS UPLOAD
---------------------
   What it does: Students can upload their DegreeWorks PDF to get
   personalized academic information in the chatbot.

   What we fixed:
   - The "Connect DegreeWorks" button wasn't opening the file picker - FIXED
   - PDF files weren't being read properly - FIXED by adding backup methods
   - Student classification was showing "Freshman" incorrectly - FIXED
   - Credit hours were showing wrong numbers - FIXED
   - Advisor name was showing random words - FIXED

   How it works now: When you upload your DegreeWorks PDF, the system reads
   your credits, calculates your year (Freshman/Sophomore/Junior/Senior),
   and finds your advisor's name correctly.


5. CONTACT SUPPORT FEATURE
--------------------------
   What it does: Students can now report bugs or request features directly
   from the app.

   Where to find it: In the sidebar (left side), there's a "Contact Support"
   button above the Dark Mode and Install App buttons.

   What you can do:
   - Choose a category: Bug Report, Feature Request, or Question
   - Write a subject line
   - Describe your issue or suggestion
   - Attach a file (like a screenshot) if needed
   - Submit the ticket

   After submitting: You'll see a success message, and your ticket is saved
   for the admin team to review.


6. ADMIN DASHBOARD
------------------
   What it does: Admins can now see and manage all support tickets submitted
   by students.

   Features for admins:
   - See how many tickets are open, in progress, or resolved
   - Filter tickets by status
   - View full ticket details including who submitted it
   - Mark tickets as "In Progress" or "Resolved"
   - See user email for each ticket

   How to access:
   1. Log in as admin (admin@test.com / admin)
   2. Go to Profile page
   3. Click "Open Admin Dashboard" in the Admin Access section


7. UI/UX IMPROVEMENTS
---------------------
   Changes made:
   - Admin Dashboard has icons on stat cards
   - Admin Dashboard shows who submitted each ticket
   - Back button on Admin Dashboard for easy navigation
   - Admin access moved from navbar to Profile page settings
   - Mic button glows when listening
   - Better tooltips throughout the app
   - Consistent dark mode support everywhere

HOW TO TEST THE NEW FEATURES
============================

Test Voice Input:
   1. Log in to your account
   2. Click the microphone button (next to Send button)
   3. Speak your question
   4. Check if text appears in input box
   5. Press Send

Test Chat Memory:
   1. Ask the bot something specific (e.g., "What are the prerequisites for COSC 350?")
   2. Then ask a follow-up (e.g., "What about for COSC 450?")
   3. The bot should understand you're still talking about prerequisites

Test DegreeWorks Upload:
   1. Log in to your account
   2. Go to your Profile page
   3. Click "Connect DegreeWorks"
   4. Select your DegreeWorks PDF
   5. Check if your credits, classification, and GPA show correctly

Test Contact Support:
   1. Log in to your account
   2. Open the sidebar (click the logo/hamburger icon)
   3. Click "Contact Support"
   4. Fill out the form and submit
   5. You should see a success message

Test Admin Features:
   1. Log in with admin@test.com / admin
   2. Go to Profile page
   3. Click "Open Admin Dashboard" in Admin Access section
   4. Click "Support Tickets" tab
   5. You should see any tickets that have been submitted
