FEATURE: Frequently Asked Questions from Real Usage
---------------------------------------------------
Welcome screen shows dynamically generated suggestions based on what
students are actually asking.

BACKEND ENDPOINT: GET /api/popular-questions
  - Queries chat_history for user messages
  - Filters out greetings and short messages
  - Counts frequency of similar questions
  - Returns top 5 questions asked 2+ times
  - Falls back to defaults if insufficient data

IMPLEMENTATION:
  @app.get("/api/popular-questions")
  async def get_popular_questions(db: Session = Depends(get_db)):
      # Query recent user messages
      messages = db.query(ChatHistory)\
          .filter(ChatHistory.role == "user")\
          .order_by(ChatHistory.timestamp.desc())\
          .limit(1000).all()

      # Filter and count
      questions = {}
      for msg in messages:
          text = msg.content.strip()
          if len(text) > 15 and text.endswith("?"):
              # Normalize and count
              key = text.lower()
              questions[key] = questions.get(key, 0) + 1

      # Return top questions
      sorted_q = sorted(questions.items(), key=lambda x: -x[1])
      return {"questions": [q[0] for q in sorted_q[:5] if q[1] >= 2]}

FEATURE: Microphone Button for Voice Input
------------------------------------------
Users can click mic button to speak their question instead of typing.

IMPLEMENTATION (Chatbox.jsx):
  const handleVoiceInput = () => {
    // Stop if already listening
    if (isListening) {
      recognitionRef.current?.stop();
      setIsListening(false);
      return;
    }

    const SpeechAPI = window.SpeechRecognition || window.webkitSpeechRecognition;
    if (!SpeechAPI) {
      alert("Speech recognition not supported. Try Chrome or Edge.");
      return;
    }

    const rec = new SpeechAPI();
    rec.lang = "en-US";
    rec.continuous = false;
    rec.interimResults = false;

    rec.onresult = (e) => {
      const transcript = e.results[0][0].transcript;
      setInput(transcript);
      setIsListening(false);
      inputRef.current?.focus();
    };

    rec.onerror = (e) => {
      setIsListening(false);
      if (e.error !== "no-speech" && e.error !== "aborted") {
        alert("Voice input error: " + e.error);
      }
    };

    rec.start();
  };

PROBLEM: PDF Upload Button Not Working
--------------------------------------
- Issue: File picker dialog not opening
- Solution: React ref-based approach with useRef() and onClick handler

    }
  };

PROBLEM: PDF Text Extraction Failing
------------------------------------
- Issue: pypdf extracting 0 characters from some PDFs
- Solution: Multi-library fallback extraction chain

Extraction Order (main.py):
  1. pypdf (PyPDF2 replacement) - Primary method
  2. pdfplumber - Secondary fallback
  3. PyMuPDF (fitz) - Third fallback
  4. Raw text extraction - Final fallback

PROBLEM: Classification Showing "Freshman" Incorrectly
------------------------------------------------------
- Issue: Regex matching "Freshman" from course names
- Solution: Calculate classification from earned credits

Classification Logic:
  if credits >= 90: classification = "Senior"
  elif credits >= 60: classification = "Junior"
  elif credits >= 30: classification = "Sophomore"
  else: classification = "Freshman"

PROBLEM: Advisor Name Showing Wrong Words
-----------------------------------------
- Issue: Parser matching words like "Prerequisite"
- Solution: Implemented blocklist of excluded words
